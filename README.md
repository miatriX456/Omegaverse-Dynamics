# Omegaverse Dynamics — расширение для SillyTavern

Полноценный движок для ролевых игр в Омегаверсе. Управляйте статусами, циклами, запахами и отношениями персонажей.

## Возможности
- Принудительный режим Омегаверса (даже для обычных ботов)
- Настройка статуса (Альфа/Бета/Омега) и роли
- Система запахов с категориями и готовыми шаблонами
- Календарь и циклы (течка/гон)
- Управление NPC, метками, связями
- Рандомизация всех параметров
- Экспорт/импорт данных

## Установка
Скопируйте папку расширения в `public/scripts/extensions/third-party/` и перезагрузите SillyTavern.
